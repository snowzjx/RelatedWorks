RelatedWorks

A native macOS literature manager for CS researchers.
Organize papers, annotate with cross-references, and generate
Related Works sections — powered by local AI via Ollama.

  • ✦ Project-based workspaces — one project per paper you're writing
  • ✦ PDF import with AI metadata extraction (title, authors, abstract)
  • ✦ DBLP + arXiv search with automatic BibTeX fetching
  • ✦ Semantic IDs and @cross-reference annotations
  • ✦ One-click Related Works generation in LaTeX via Ollama
  • ✦ Full Terminal UI — keyboard-driven, works over SSH
  • ✦ Deep link support via relatedworks:// URIs
⬇ Download Latest ⌥ View Source

Requires macOS 13+ and Ollama running locally.

🖥 Terminal UI

RelatedWorks ships a full interactive TUI — perfect for keyboard-driven workflows, SSH sessions, or when you prefer staying in the terminal. The TUI shares all data with the GUI in real time.

$ swift run RelatedWorks
