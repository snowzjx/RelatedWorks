RelatedWorks

A native macOS & iOS academic literature manager for Computer Science researchers.
Organize papers, take interconnected notes, and automatically draft
Related Works sections — powered by local or cloud AI.

  • ✦ Project-based workspaces — one project per paper you're writing
  • ✦ PDF import with AI metadata extraction (title, authors, abstract)
  • ✦ DBLP + arXiv search with automatic BibTeX fetching
  • ✦ Semantic IDs and @cross-reference annotations
  • ✦ One-click Related Works generation in LaTeX via AI
  • ✦ Multiple AI backends — Ollama (local) and Google Gemini
  • ✦ Export & import projects as .relatedworks files
  • ✦ Full Terminal UI — keyboard-driven, works over SSH
  • ✦ Live search & filter — search by ID, title, authors, venue, year, abstract, or annotation with match highlighting
  • ✦ Deep link support via relatedworks:// URIs
