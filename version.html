Version History

A concise release log for the public site.

Source changelog: version.md

v2.1.8

Latest release. Related Works generation now streams output as it is produced instead of waiting for the full draft to finish. The macOS About window also includes a link back to the RelatedWorks website.

v2.1.7

Previous release. Stabilized generation settings and polished localization. Preserves the selected Ollama backend across generation workflows, decouples per-project prompt fallback behavior from shared settings, avoids synchronous store loading in app flows, standardizes annotation localization text, and unifies the Related Works generation implementation.

v2.1.6

Previous release. Added a dedicated citation graph window for inspecting project references and shared outside references, source-aware reference metadata for citation comparisons, and Hookmark scripting integration for linking RelatedWorks items into external knowledge workflows.

v2.1.5

Previous release. Added a bulk Inbox import workflow for processing multiple captured PDFs and read-only AppleScript support for inspecting projects, papers, and current selections from automation tools.

v2.1.4

Previous release. Exported .relatedworks projects can now be opened directly from Finder, and repeated Finder open requests are deduplicated so the same project is not opened multiple times.

v2.1.3

Previous release. Dragging a PDF onto the paper list now opens Add Paper with that PDF already loaded in the drop zone. The existing metadata extraction and semantic ID flow stays intact, while the extra browsing step is removed.

v2.1.2

Previous release. Improved generation failure handling and timeout clarity for Ollama workflows. Added configurable Ollama request timeout in Settings, friendlier timeout and failure messages during Related Works generation, removed fallback template drafts from error responses, and added missing localization entries for new timeout and backend status strings.

v2.1.1

Previous release. Guided first-launch onboarding with clearer step-by-step tutorial bubbles. Better tutorial copy and localization for the onboarding flow. Visible export progress feedback for project exports. Dedicated version history page to keep the home page focused.

v2.1.0

Previous release. Chinese (Simplified) localization for the macOS and iOS app interface. In-app language switcher on macOS for English, Chinese (Simplified), or Follow System. On-demand iCloud PDF download on macOS with automatic opening after sync.

v2.0.0

Foundation release. PDF Inbox via Share Extension on iOS and macOS for Safari and Files sharing. Per-project generation prompts with Survey, Research Paper, Tech Report, and Custom presets. Liquid Glass search box styling on macOS.
